Use it to finish the 30-day plan first, then decide whether secondary tracks have room.

## Daily Operating Order

Use the main SOP first:

1. Pick today's objective from the 30-day plan.
2. Check existing replies and signals.
3. Prepare or publish one main asset.
4. Do targeted interactions.
5. Record metrics, links, learnings, and next action.

See: [SandBase Daily Growth SOP](playbooks/sandbase-daily-growth-sop.md)

## Daily Time Budget

| Track | Time | Outcome |
|-------|------|---------|
| 30-day growth foundation | 60-120 min | Finish today's 30-day-plan output and log it |
| Overseas builders outreach | 20-45 min | Optional: find leads, send useful messages, learn from builders |
| Hackathon sponsorship | 20-45 min | Optional: find events, contact organizers, prepare support offers |

If time is limited, use this priority:

1. Reply to existing conversations.
2. Finish today's 30-day-plan output.
3. Do one high-quality outreach action only if the main output is done.
4. Record what happened.

## Rules

- No mass outreach.
- No private account details in public logs.
- No publishing private conversations without permission.
- No paid sponsorship or directory submission without explicit approval.
- Public content should help builders, not just promote SandBase.
