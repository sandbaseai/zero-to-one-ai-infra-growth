# Directory Submission Kit

Status: Ready for first controlled submission

Owner: SandBase growth

## Purpose

This kit keeps external directory submissions consistent.

The goal is not to submit SandBase everywhere. The goal is to publish a small number of credible profiles that reinforce SandBase as agent infrastructure for developers building production AI agents.

## Submission Rule

```text
One credible profile is better than ten generic AI tool listings.
```

Before submitting, check:

1. Does the directory have a real developer or startup audience?
2. Can SandBase be categorized as infrastructure, developer tools, or AI infrastructure?
3. Can the profile link to the website, GitHub, docs, blog, Discord, or LinkedIn?
4. Does the page look like a long-term trust asset rather than a spam directory?
5. Can the listing avoid sounding like a generic AI app?

If the answer is no, skip it for now.

## Cost Rule

Do not pay for directory submissions by default.

Budget:

```text
Maximum $30 per directory / launch action.
Free is preferred.
Anything paid still requires human confirmation before purchase or submit.
```

For every channel, record:

- free or paid
- whether a free listing exists
- whether paid placement is optional
- whether the paid option is worth considering later
- whether the paid option is within the $30 limit
- whether pricing is unconfirmed

If pricing is unclear, stop before final submission and confirm manually. If a paid option is above $30, do not use it.
